Таблица script - основной элемент управления плагинами, который неявно создается перед запуском скрипта, создавать его не нужно

Все Lua-файлы в плагине вызываются во время  во время определенного события, во фреймворке szlua они имеют названия szlua events
Например события startup срабатывает во время запуска приложения, нечего не возвращает и приложение не ждет завершения этого события

Для того, чтобы подписаться на события, нужно создать таблицу events внутри таблицы script
